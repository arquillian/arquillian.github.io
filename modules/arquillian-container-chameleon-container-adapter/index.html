Arquillian Chameleon Container Adapter

Arquillian Chameleon Container Adapter

Chameleon container that dynamically loads correct container adapter versions

What is Arquillian Container Chameleon?

The Chameleon container can quickly adapt to your needs of testing against several containers (JBoss AS / JBoss EAP / WildFly / GlassFish) or even switching between different modes (Managed / Remote / Embedded) without requiring additional dependency configuration.

Artifacts

Build dependency artifacts

Add the appropriate stanza to the <dependencies> element in your pom.xml:

Arquillian Chameleon Container Adapter

<dependency>
  <groupId>org.arquillian.container</groupId>
  <artifactId>arquillian-container-chameleon</artifactId>
  <version>1.0.0.CR2</version>
  <scope>test</scope>
</dependency>
Warning: You'll also need to add the Arquillian core platform and possibly additional dependencies. Please refer to the guides for more details.
