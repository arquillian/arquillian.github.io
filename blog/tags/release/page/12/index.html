Arquillian Container Jetty 1.0.0.CR1 Released

Posted by Aslak Knutsen on Jun 29, 2011
Jun 29
| Comments

The Arquillian team is proud to announce the 1.0.0.CR1 release of the Arquillian Container Jetty component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian Container Jetty
Modules
  • Arquillian Jetty Embedded 6.1.x Container Adapter
  • Arquillian Jetty Embedded 7.x and 8.x Container Adapter
Version 1.0.0.CR1 view tag
Release date 2011-06-29
Released by Aslak Knutsen
Compiled against
  • Arquillian Core – 1.0.0.CR1
