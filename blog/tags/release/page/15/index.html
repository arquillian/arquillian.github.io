Arquillian Container Tomcat 1.0.0.CR1 Released

Posted by Aslak Knutsen on Jun 28, 2011
Jun 28
| Comments

The Arquillian team is proud to announce the 1.0.0.CR1 release of the Arquillian Container Tomcat component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian Container Tomcat
Modules
  • Arquillian Tomcat Managed 5.5.x Container Adapter
  • Arquillian Tomcat Embedded 6.x Container Adapter
  • Arquillian Tomcat Remote 6.x Container Adapter
  • Arquillian Tomcat Managed 6.x Container Adapter
  • Arquillian Tomcat Embedded 7.x Container Adapter
  • Arquillian Tomcat Managed 7.x Container Adapter
Version 1.0.0.CR1 view tag
Release date 2011-06-28
Released by Aslak Knutsen
Compiled against
  • Arquillian Core – 1.0.0.CR1
