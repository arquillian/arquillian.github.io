Arquillian Core 1.0.0.Beta2 Released

Posted by Aslak Knutsen on Jun 08, 2011
Jun 08
| Comments

The Arquillian team is proud to announce the 1.0.0.Beta2 release of the Arquillian Core component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian Core
Version 1.0.0.Beta2 view tag
Release date 2011-06-08
Released by Aslak Knutsen
Compiled against
  • JUnit – 4.8.1
  • ShrinkWrap Core – 1.0.0-beta-2
  • ShrinkWrap Descriptors – 1.0.0-beta-1
  • TestNG – 5.14.6

Thanks to the following list of contributors: Thomas Diesler, Aslak Knutsen
