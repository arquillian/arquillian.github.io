Arquillian OSGi 2.1.0.CR14 Released

Posted by Thomas Diesler on Mar 25, 2014
Mar 25
| Comments

The Arquillian team is proud to announce the 2.1.0.CR14 release of the Arquillian OSGi component!

What is Arquillian?

Arquillian is open source software that empowers you to test JVM-based applications more effectively. Created to defend the software galaxy from bugs, Arquillian brings your test to the runtime so you can focus on testing your application's behavior rather than managing the runtime. Using Arquillian, you can develop a comprehensive suite of tests from the convenience of your IDE and run them in any IDE, build tool or continuous integration environment.

Release details

Component Arquillian OSGi
Modules
  • Arquillian OSGi :: Container :: JBoss :: Embedded
  • Arquillian OSGi :: Container :: Felix :: Embedded
  • Arquillian OSGi :: Container :: Equinox :: Embedded
  • Arquillian OSGi :: Container :: Karaf :: Embedded
  • Arquillian OSGi :: Container :: Karaf :: Managed
  • Arquillian OSGi :: Container :: Karaf :: Remote
